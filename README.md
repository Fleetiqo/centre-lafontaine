## 🚀 Développement

```bash
# Installer les dépendances
npm install

# Démarrer le serveur de développement
npm run dev

# Build pour production
npm run build

# Prévisualiser le build
npm run preview
```

## 📦 Déploiement

Le site est configuré pour être déployé sur GitHub Pages avec GitHub Actions.

1. Push vers la branche `main`
2. GitHub Actions build et déploie automatiquement
3. Le site sera disponible à: `https://[username].github.io/centre-lafontaine/`
